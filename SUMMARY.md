# 📋 ملخص المشروع - Booking4U

## 🎯 نظرة عامة

**Booking4U** هو نظام حجز مواعيد ذكي ومتطور مصمم خصيصاً للسوق السعودي. يوفر النظام حلولاً شاملة لإدارة الحجوزات للعيادات الطبية، صالونات التجميل، المراكز الرياضية، المطاعم، والعديد من الخدمات الأخرى.

## 🛠️ التقنيات المستخدمة

### Backend
- **Node.js** - بيئة تشغيل JavaScript
- **Express.js** - إطار عمل الويب
- **MongoDB** - قاعدة بيانات NoSQL
- **Mongoose** - ODM لـ MongoDB
- **JWT** - مصادقة آمنة
- **bcryptjs** - تشفير كلمات المرور
- **express-validator** - التحقق من صحة البيانات
- **multer** - رفع الملفات
- **nodemailer** - إرسال البريد الإلكتروني
- **twilio** - إرسال SMS و WhatsApp

### Frontend
- **React.js** - مكتبة واجهة المستخدم
- **Tailwind CSS** - إطار عمل CSS
- **React Router** - إدارة التنقل
- **React Hook Form** - إدارة النماذج
- **Zustand** - إدارة الحالة
- **React Query** - إدارة البيانات
- **React Hot Toast** - الإشعارات
- **Axios** - طلبات HTTP

### Infrastructure
- **Docker** - حاويات التطبيق
- **Docker Compose** - إدارة الخدمات
- **Nginx** - خادم الويب العكسي
- **MongoDB** - قاعدة البيانات

## 🔧 أوامر التشغيل السريعة

### باستخدام Makefile
```bash
make help          # عرض جميع الأوامر المتاحة
make install       # تثبيت جميع التبعيات
make dev           # تشغيل في وضع التطوير
make docker-up     # تشغيل باستخدام Docker
make docker-down   # إيقاف التطبيق
make docker-logs   # عرض سجلات Docker
make health        # فحص صحة التطبيق
```

### باستخدام سكريبت التشغيل
```bash
./start.sh start   # تشغيل التطبيق
./start.sh stop    # إيقاف التطبيق
./start.sh restart # إعادة تشغيل التطبيق
./start.sh logs    # عرض السجلات
./start.sh health  # فحص الصحة
./start.sh clean   # تنظيف وإعادة البناء
```

## 📚 نقاط النهاية الرئيسية للـ API

### المصادقة
- `POST /api/auth/register` - تسجيل مستخدم جديد
- `POST /api/auth/login` - تسجيل دخول
- `GET /api/auth/me` - معلومات المستخدم الحالي
- `PUT /api/auth/profile` - تحديث الملف الشخصي

### الحجوزات
- `POST /api/bookings` - إنشاء حجز جديد
- `GET /api/bookings/business/:businessId` - حجوزات الأعمال
- `GET /api/bookings/my-bookings` - حجوزاتي
- `PUT /api/bookings/:bookingId/status` - تحديث حالة الحجز
- `PUT /api/bookings/:bookingId/cancel` - إلغاء الحجز

### الخدمات
- `GET /api/services/:businessId` - خدمات الأعمال
- `POST /api/services` - إضافة خدمة جديدة
- `PUT /api/services/:serviceId` - تحديث خدمة
- `DELETE /api/services/:serviceId` - حذف خدمة

### الأعمال
- `GET /api/businesses` - قائمة الأعمال
- `POST /api/businesses` - إنشاء عمل جديد
- `GET /api/businesses/:businessId` - تفاصيل العمل
- `PUT /api/businesses/:businessId` - تحديث العمل

## 🎯 حالات الاستخدام

### للعملاء
1. **التسجيل**: إنشاء حساب جديد
2. **البحث**: البحث عن الأعمال والخدمات
3. **الحجز**: اختيار الخدمة والموعد
4. **التأكيد**: تأكيد الحجز والدفع
5. **المتابعة**: متابعة الحجوزات والتقييمات

### لأصحاب الأعمال
1. **إعداد الحساب**: إنشاء حساب صاحب عمل
2. **إدارة الأعمال**: إضافة وتعديل معلومات العمل
3. **إدارة الخدمات**: إضافة وتعديل الخدمات المقدمة
4. **إدارة الحجوزات**: عرض وإدارة الحجوزات
5. **التقارير**: عرض الإحصائيات والتقارير

## 💰 نموذج الأعمال

### باقات SaaS
- **الباقة الأساسية (99 ريال/شهر)**
  - حجوزات غير محدودة

  - إدارة العملاء

  - تقارير متقدمة
